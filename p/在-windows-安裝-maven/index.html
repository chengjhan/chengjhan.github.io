在 Windows 安裝 Maven

使用 Maven 的第一步

Apr 06, 2023
1 minute read

安裝 JDK

在安裝 Maven 之前必須先安裝 JDK (Java Development Kit)，本篇以安裝 JDK 1.8 為例，至 Oracle 官方網站下載並安裝。

下載 64 位元的安裝版，依對話框安裝即可。

安裝後必須新增環境變數：
至「本機」> 右鍵「內容」>「進階系統設定」>「環境變數」>「新增系統變數」設定。

變數名稱：JAVA_HOME。
變數值：C:\Program File\Java\jdk1.8.0_351 (預設安裝路徑)。

編輯 Path 環境變數：

新增 Path：
在 Path 變數新增「%JAVA_HOME%\bin」。

確認安裝結果：

java -version

安裝 Maven

至 Apache Maven 官方網站下載 Maven 解壓縮放置於欲放置的目錄，本篇放置於 D:\ 目錄下。

必須新增環境變數：
至「本機」> 右鍵「內容」>「進階系統設定」>「環境變數」>「新增系統變數」。

變數名稱：M2_HOME。
變數值：D:\apache-maven-3.8.1。

新增 Path：
在 Path 變數新增「%M2_HOME%\bin」。

確認安裝結果：

mvn -v

maven java
